# 高考英语趋势分析 (2010-2022) — 4.3 真模型

> 用 stdlib `statistics` (零新增依赖) 跑线性回归, **非押题**, 只做趋势识别.
> 数据源: GAOKAO-Bench 镜像 (`exam_questions` 334 题), 全标辽宁卷型.

---

## 1. 题型占比年趋势 (slope 正 = 占比逐年上升)

| 题型 | slope / 年 | 趋势 | 均占比 |
|---|---|---|---|
| 阅读理解 | **+0.028** | ⬆ 上升 | 41.1% |
| 完形填空 (七选五/语篇) | +0.012 | ⬆ 上升 | 8.6% |
| 完形填空 | +0.011 | ⬆ 上升 | 9.3% |
| 短文改错 | +0.011 | ⬆ 上升 | 9.3% |
| 语法填空 | +0.008 | ⬆ 上升 | 10.1% |
| **单选 (语法/词汇)** | **-0.069** | ⬇ **下降** | 21.5% |

**结论 1**: 单选 (传统语法选择题) 大幅下降 (5 年内已基本消失), 阅读理解持续走高占近半. 教学侧重应**强化阅读 + 完形/语法填空**, 单选题型只作复习辅助.

## 2. 词汇量年增长

| | 值 |
|---|---|
| slope (实义 token / 年) | **+99.57** |
| 解读 | **词汇量逐年膨胀** |
| 13 年总 token | 2010: 2584 → 2022: 2582 (波动, 整体趋势上升) |
| 峰值 | 2018: 3887 (近年最难) |

**结论 2**: 高考阅读题文长在上升, 学生**词汇量要求逐年提升**. 教材覆盖 ~2000 词不够, 需 HV_extra (15→287) 类高价值课外词补充.

## 3. 近 3 年新兴 / 高频上升词 (top)

| word | 近 3 年频次 | 老时期频次 | 上升倍数 |
|---|---|---|---|
| drones | 17 | 0 | 18× |
| nutria | 16 | 0 | 17× |
| cardiff | 13 | 0 | 14× |
| rail | 12 | 1 | 6.5× |
| genius | 12 | 0 | 13× |

**结论 3**:
- **drones / rail** = 科技进步主题 (人与社会/科学与技术)
- **cardiff** = 英国地名 (人与社会/历史、社会与文化)
- **nutria** = 动物保护主题 (人与自然/自然生态)
- 教材若不覆盖科技 + 文化主题, 学生考阅读 reading 容易"看不懂"

## 模型说明 (非押题, 治理 §3.3 兼容)

- 用 `statistics.mean` + 简单线性回归 (stdlib only)
- 不预测下年题面 (banned)
- 输出趋势 slope + 解读, 给教学侧 "**该多教什么**" 而非"该考什么"
- 计算源: `backend/services/trend/model.py` (`question_type_year_trend` / `vocab_year_growth` / `top_rising_words`)
- API: `/api/trend/question_type_trend` / `/api/trend/vocab_growth` / `/api/trend/rising_words`

后续可扩 (留 P2):
- ARIMA / Prophet 时间序列 (需装 statsmodels)
- 主题语义 embedding 趋势 (需装 sentence-transformers)
- 难度趋势 (待学生答题日志)
